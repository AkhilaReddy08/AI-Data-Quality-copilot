# 📊 AI Data Quality Copilot

An AI-powered data quality analysis tool built using Python and Streamlit.

## 🚀 Features
- Upload CSV file
- Detect missing values
- Detect duplicate rows
- Identify outliers
- Validate email formats
- AI-generated insights
- Data quality health score
- Interactive dashboard with charts
- Downloadable report

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

## 📸 Project Flow
1. Upload dataset
2. Automatic data quality analysis
3. AI insights generated
4. Health score calculated
5. Visual dashboard displayed

## 📊 Output Example
- Data Quality Report (JSON)
- Health Score (0–100)
- AI Recommendations

## ▶️ Run Project
```bash
streamlit run app.py
